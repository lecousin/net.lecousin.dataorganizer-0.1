Version 0.2.0
Some improvements have been added since the first version:
- Data images: you can now remove some, edit them...
- Menu 'Files information' on a data that displays physical files linked to the data
- You can now choose a system application to open a data instead of only the default one
- Several display improvements and issues fixed
- Possibility to create data list (corresponding to a selection for instance)
- Files types support improvement:
- ASF(WMA and WMV) files are well detected (WMV files no more detected as audio files)
- NeufBox Media Center list is now saved and restored when re-launched
