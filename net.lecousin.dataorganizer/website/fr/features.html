Ecran principal

 L'application présente une fenêtre principalement composée de 3 parties:

  • A gauche: un ensemble de critères de recherche vous permet de filtrer les données affichées
  • Au centre, en haut: la liste de vos données, filtrée selon les critères de recherche
  • Au centre, en bas: les informations concernant la donnée sélectionnée

Recherche

 La fenêtre de recherche vous permet de filtrer les données affichées.
Pour la plupart des critères, vous pouvez soit demander d'inclure les données répondant à ce critère, ou vous pouvez inverser le critère et ainsi demander d'exlure les données répondant à ce critère. Pour ce faire il suffit d'utiliser le bouton d'inversion:
